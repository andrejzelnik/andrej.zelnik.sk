# andrej.zelnik.sk

Personal landing page at [andrej.zelnik.sk](https://andrej.zelnik.sk). Plain HTML/CSS, no build step.

## Preview

Open `index.html` directly in a browser.

## Deploy

Push to `main` — GitHub Actions deploys to GitHub Pages automatically.

To enable on a fresh fork: **Settings → Pages → Source → GitHub Actions**.

## Favicons

Regenerate at [favicon.io](https://favicon.io/favicon-generator/): text `a`, Fugaz One, size 125, white `#fff` on black `#000` rounded.
